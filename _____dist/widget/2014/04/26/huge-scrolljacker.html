Huge.Slider

Full page scroll jacking slider that's touch friendly

Prereqs

Compatibility

  • IE9+
  • Chrome, Opera, lynx

Usage

$(document).ready(function() {
        new Slider();
  });

section {
    position: relative;
}
html, body, article {
    height: 100%;
}

html, body {
    overflow: hidden;
}

Options

Name type default description
animation boolean or the string 'static' true Includes a modal-backdrop element. Alternatively, specify static for a backdrop which doesn't close the modal on click.
