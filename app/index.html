Pixels To Ems

name
huge-em
type
function
arguments
pixels
px value you want to convert to ems
base
base px value to convert the ems off of

Example

SCSS
div#example {
// first pair
&.m-1 { width: huge-em(300); }
&.m-2 { width: 300px; }
// second pair
&.m-3, &.m-4 { font-size: 20px; }
&.m-3 { width: huge-em(400, 20); }
&.m-4 { width: 400px; }
}
