General

Circle with Arrow

Name:
circle-with-arrow
Arguments
Size:
Size of the circle you want
Circle color:
Color of the circle
Arrow color:
Arrow color you want (durr)

Example

Code:
{% highlight scss %}@include circle-with-arrow(50px, $magenta, #000);{% endhighlight %}

Card Flip

name
card-flip
Arguments
Direction:
Direction you want it to flip, left right up down (corner is in there but hasn't been tested in IE, but who cares AMIRIGHT?)
Height:
Card height
Width:
Card width
Perspective:
Set the animation perspective
Duration:
The durations of the animation

Example

Code:
{% highlight scss %}@include card-flip(left, 200px, 200px, 1000, 1s);{% endhighlight %}
