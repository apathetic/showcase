Huge Inc. Nav.

    About

    This is an abstraction of the menu used on the Huge Inc. site. It is a fullscreen menu, with a subtle animation on the menu items when they fade in. In fact, there are several improvements over the original implementation. Namely, no javascript is used to calculate the height / spacing of the menu items when viewed on different screen-sizes, orientations. Welcome the new vh and vw css selectors*

    * not valid in IE8
